ACL 简介

ACL—访问控制列表（Access Control List），为Linux文件系统提供了更加灵活的权限机制。可以针对某一个用户或某一个群组来设定特定的权限需求。

查看文件系统是否支持 ACL

两个命令

setfacl

设置文件或目录的 ACL 权限

  • 为 user 设置 ACL

  • 删除 user 的 ACL 设置

  • 为 group 设置 ACL

  • 删除 group 的 ACL 设置

  • 删除文件或目录的全部 ACL 权限

getfacl

获取文件或目录的 ACL 权限
